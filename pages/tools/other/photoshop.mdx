# Photoshop Basics

[Photoshop](https://www.adobe.com/products/photoshop.html) is a powerful image editing software that allows you to create and manipulate digital images.

To get started with Photoshop, you will need to have the software installed on your computer. Once you have installed and opened Photoshop, you can begin exploring its features.

## Creating a New Project

To create a new project, click on **File** in the top menu, then select **New**. Here, you can set the dimensions and resolution of your image, as well as other settings such as color mode and background contents.

File > New

![Creating a new project](https://i.imgur.com/2Kv7KbJ.png)

## Importing Images

To import an image into your project, click on **File** in the top menu, then select **Open**. Choose the image you want to import from your file browser and click **Open**.

Alternatively, you can drag and drop an image into your project from your file browser.
